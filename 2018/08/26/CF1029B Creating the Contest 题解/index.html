CF1029B Creating the Contest

Description

给定一个序列a=a1,a2,…,aia=a_{1},a_{2},\ldots ,a_ia=a​1​​,a​2​​,…,a​i​​，aaa序列单调递增，从中取出nnn个数后（不需要连续）组成新序列bbb，并对于任意一个从111到p−1p-1p−1的数满足bj+1≤2×bjb_{j+1}\leq 2\times b_{j}b​j+1​​≤2×b​j​​，求bbb序列长度的最大值。其中1≤n≤2×1051\leq n\leq 2\times 10^51≤n≤2×10​5​​。

Solution

首先，假设我们取了第iii个数和第i+ji+ji+j个数进入bbb序列，并且中间的数都没有取，因为题目描述，我的这两个数满足bi+j≤2×bib_{i+j}\leq 2\times b_{i}b​i+j​​≤2×b​i​​。

然后我们考虑怎么让这个长度变成最大，要想让新的序列长度变大，我们要取更多的数，而因为aaa序列是单调递增的，所以我们既然已经取了aia_{i}a​i​​和ai+ja_{i+j}a​i+j​​，为什么不考虑一下中间的数呢？我们发现，因为aaa序列是单调递增的，所以我对于任意一个ak(k∈(i,i+j))a_{k}(k\in (i,i+j))a​k​​(k∈(i,i+j))，aka_{k}a​k​​是始终小于ai+ja_{i+j}a​i+j​​的，那么，既然bi+j≤2×bib_{i+j}\leq 2\times b_{i}b​i+j​​≤2×b​i​​,所以bi+j≤2×bkb_{i+j}\leq 2\times b_{k}b​i+j​​≤2×b​k​​,因此我对于任意一个aka_{k}a​k​​也是可以取的。因此，虽然题目中强调不需要连续取数，实际上只有连续取数才能达到某个序列的最大长度。因此，我们把原数列从左到右扫一遍，定义一个cntcntcnt变量，每有一个数满足条件就自加111，不满足条件的时候就维护一个max(cnt)max(cnt)max(cnt)，最终输出这个max(cnt)max(cnt)max(cnt)即可。
