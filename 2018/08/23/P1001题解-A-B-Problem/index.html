P1001题解 A+B Problem

Description

输入两个整数$ a,b $，输出它们的和(|a|,|b|<=10^9)。

Solution

我们容易发现，因为a,b的和满足|a|,|b|<=10^9，所以可以直接相加求和输出即可。
